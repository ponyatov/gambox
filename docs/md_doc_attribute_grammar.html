attribute grammar

https://en.wikipedia.org/wiki/Attribute_grammar

  • An attribute grammar is a formal way to supplement a [[formal grammar]] with [[semantic]] information processing.
  • Semantic information is stored in attributes associated with [[syntax/terminal]] and [[nonterminal]] symbols of the [[syntax/grammar]]. The values of attributes are result of attribute evaluation rules associated with [[production]]s of the grammar. Attributes allow to transfer information from anywhere in the abstract syntax tree to anywhere else, in a controlled and formal way.[1]
